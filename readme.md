# CRAWfish - a Clash of Clans Rust API Wrapper

## Description
A rust wrapper over the Clash of Clans API. Provides structs for datatypes
used by the API, and a `Client` struct to access the API using various
methods.

Use of this library requires you to have a Clash of Clans Developer
Account from <https://developer.clashofclans.com/>, and an API token valid
for the IP address of the machine running the library.
